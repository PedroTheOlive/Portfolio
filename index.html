About Me

I’m Pedro Arruda, a Cornell University Information Science student focused on UX/UI design, web development, and data visualization. I enjoy combining design and programming to build clear, interactive, and meaningful user experiences.

My work spans visual design and full-stack development, with hands-on experience using JavaScript, Python, C++, and SQL, along with tools like Figma, Photoshop, and D3.js. I focus on usability, data clarity, and accessibility — ensuring that each project feels intuitive and informative.

Recent projects include a 2D action platformer built with LibGDX, a cocktail enthusiasts website developed with PHP and SQLite, a responsive website for the Cornell Rowing Club, and an interactive data visualization platform that explores U.S. crime statistics using D3.js.

In my crime visualization project, I developed an interactive website to explore national datasets, focusing on data interpretation, cleaning, and presentation. The work involved parsing large JSON and CSV files, structuring them for D3 visualizations, and designing an interface where users could explore meaningful patterns in crime data across regions.

Across all of my work, I aim to bridge creativity and technical precision — designing solutions that are functional, scalable, and engaging. I’m currently seeking opportunities in product design, UX/UI development, or front-end engineering where I can continue to grow as both a designer and a developer.
